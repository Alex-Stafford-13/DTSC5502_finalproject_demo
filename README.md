## This project uses a convolutional neural network (CNN) to perform a binary classification task on a dataset of signatures. Signature dataset was taken from kaggle: https://www.kaggle.com/datasets/akashgundu/signature-verification-dataset/data  
## The .ipynb file "DTSC5502_finalproject_model2" contains the code required to run the model
## The .csv file "sigs" contains the file path names of each signature and their labeled classification
## The .zip folder "signatures" contains the .jpg files of all signatures
## All three are needed to run the model. The code will call upon sigs.csv and the contents of signatures.zip
